On one of my engagements here at Microsoft, we helped a Client Partner implement several Azure Backend components related to Azure Media Services. The Client Partner's Frontend Angular application requried the Azure Media Player to send telemetry information to Azure Application Insights.

In this blog I want to give a brief overview of the sample code I wrote that demonstrates how to send Azure Media Player telemetry information to Azure Application Insights.

You can find the code used in this blog on Github here

Prerequisites!

  • Access to the Azure Portal to view Telemetry information
  • Angular 7
  • Application Insights Asset and the Instrumentation Key


Overview!

The sample code is a basic Angular 7 Application that uses the Azure Media Player to stream sample media and send the telemetry information to Azure Application Insights. For brevity, please review the README.md file, which provides a detailed breakdown of the AmpDiagnosticsLoggerService, which is a custom TypeScript library I created based upon the amp-diagnosticsLogger.js JavaScript library that is responsible for sending Azure Media Player telemetry informaton to Azure Application Insights.

app.component.ts

  • In *ngOnInit*, creates a list of custom mezzanine objects, which is just a custom interface with hardcoded values.
  • In *ngAfterViewInit*, initializes the Azure Media Player that is embedded in app.component.html
  • In *ngAfterViewInit*, configures the AmpDiagnosticsLoggerConfiguration object and passes the object to the "Initialize" method in the AmpDiagnosticsLoggerService Service
    • . See README.md file for details on the AmpDiagnosticsLoggerConfiguration and AmpDiagnosticsLoggerService classes
  • setMovie method is called whenever the user clicks to play a movie. When *setMovie* is called, the *log* method of the AmpDiagnosticsLoggerConfiguration service is called and the *mezzanine* id is passed. This id will be set to the Application Insights User Id field.


app.component.html

The html template for app.component.ts displays an Azure Media Player and loop through the list of mezzanine objects and displays them on the screen. When you click on either the Play picture or url, the setMovie method will be called and will be passed the current mezzanine object. The video should start playing and Azure Media Player telmetry will be sent to Application Insights.
