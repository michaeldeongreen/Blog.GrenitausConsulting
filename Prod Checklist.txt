New PC:
1) In the gc-cli appsettings.{environment}.json, ensure that the directory paths are correct

2) Re-compile gc-cli with: dotnet publish -c Release -r win10-x64

3) Modify publish.sh directory paths

4) Run ./publish.sh

Also note that you need to navigate to the gc-cli directory to build in dev: 
Git\Blog.GrenitausConsulting\Core.cli\src\gc-cli\bin\Release\netcoreapp2.0\win10-x64\publish
